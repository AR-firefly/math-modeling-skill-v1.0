# 数学建模算法选型指南

## 选型决策树

```
问题规模小（n ≤ 10-15）
  └─ 可以用精确算法
       ├─ TSP → Held-Karp DP
       ├─ 指派 → 匈牙利算法
       └─ 线性规划 → 单纯形/内点法

问题规模中等（n ≤ 100）
  ├─ 确定性问题 → 分支定界 / 割平面
  └─ NP-hard问题 → 启发式方法
       ├─ 有明确规则 → 构造式启发式（贪心/FFD/LPT）
       └─ 无明确规则 → 元启发式

问题规模大（n > 100）或需多目标
  └─ 元启发式
       ├─ 单目标 → GA / SA / TS
       ├─ 多目标 → NSGA-II / MOEA/D / SPEA2
       └─ 带约束 → GA+惩罚函数 / ε-约束法
```

---

## 常见题型-算法匹配

| 题型        | 推荐算法             | 备选方案              |
| ----------- | -------------------- | --------------------- |
| TSP路径规划 | Held-Karp DP（精确） | LKH / 最近邻基线      |
| VRP/VRPTW   | GA + 2-opt           | 禁忌搜索 / 蚁群       |
| 多目标优化  | NSGA-II + VNS        | MOEA/D / SPEA2        |
| 连续优化    | 粒子群 / 差分进化    | 梯度下降 / 贝叶斯优化 |
| 组合优化    | 遗传算法             | 模拟退火 / 禁忌搜索   |
| 布局优化    | 两阶段（粗搜+精搜）  | 贝叶斯优化            |

## 参数设定参考

### 遗传算法（GA）

| 参数     | 推荐值      | 说明             |
| -------- | ----------- | ---------------- |
| 种群规模 | 100-200     | 问题大取上限     |
| 交叉概率 | 0.8-0.9     | PMX/顺序交叉     |
| 变异概率 | 0.05-0.15   | 自适应更佳       |
| 精英保留 | 4-6个       | 防退化           |
| 终止条件 | 200代无改进 | 或最大500-1000代 |

### NSGA-II

| 参数     | 推荐值  | 说明               |
| -------- | ------- | ------------------ |
| 种群规模 | 100     | 多目标不宜过大     |
| 交叉概率 | 0.9     | 保持多样性         |
| 变异概率 | 0.15    | 配合局部搜索可降低 |
| VNS间隔  | 5-10代  | 对F1前沿执行       |
| 迭代次数 | 200-500 | 以前沿收敛为准     |

### VNS邻域结构

| 序号 | 操作       | 搜索范围 |
| ---- | ---------- | -------- |
| N₁   | 2节点交换  | 小扰动   |
| N₂   | 节点插入   | 中等扰动 |
| N₃   | 子序列反转 | 大扰动   |
| N₄   | 2-opt      | 路径优化 |
